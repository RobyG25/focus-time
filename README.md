# 🛡 מגן הפוקוס

**כלי ניהול זמן לאנשי משרד — עם דגש על התמודדות עם הפרעות**

> אפליקציית ווב פשוטה, מהירה, וללא צורך בהתקנה. כל הנתונים נשמרים מקומית בדפדפן — אין שרת, אין התחברות, אין שיתוף מידע.

---

## ✨ מה יש באפליקציה

| פיצ'ר | תיאור |
|---|---|
| ⏱ **טיימר פוקוס** | שלושה מצבים: פומודורו (25/5), עבודה עמוקה (50/10), זרימה (90/20) |
| 🔔 **יומן הפרעות** | תיעוד מהיר של כל הפרעה עם גרף דפוסים |
| 🛡 **ספר טקטיקות** | אסטרטגיות מוכחות להתמודדות עם הסחות דעת במשרד |
| ✅ **רשימת יום** | צ'קליסט בוקר וסיום יום שמתאפס אוטומטית מדי יום |
| 💾 **זיכרון מקומי** | הנתונים נשמרים בדפדפן של כל משתמש בנפרד, מתאפסים בחצות |

## 🚀 שימוש

או גש לאפליקציה החיה: **[your-username.github.io/focus-defender](https://github.com)**

## 🔒 פרטיות

- כל הנתונים נשמרים **אך ורק בדפדפן המקומי** של המשתמש (`localStorage`)
- אין שרת, אין מסד נתונים, אין analytics
- כל משתמש רואה רק את הנתונים שלו בלבד
- הנתונים מתאפסים אוטומטית בתחילת כל יום
